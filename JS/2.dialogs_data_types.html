Examples: Dialogs, Data Types, and Operators

Example 1: Using Alerts

// Displaying an alert message\nalert(\"Hello, this is an alert dialog!\");

Example 2: Using Prompt

// Getting user input using prompt\nlet name = prompt(\"What is your name?\");\nalert(\"Hello, \" + name + \"!\");

Example 3: Confirm Dialog

// Using confirm dialog\nlet isConfirmed = confirm(\"Do you want to continue?\");\nalert(\"Your choice: \" + (isConfirmed ? \"Yes\" : \"No\"));

Example 4: Data Types

// Checking data types\nlet number = 42;\nlet text = \"Hello\";\nlet isTrue = true;\nconsole.log(typeof number, typeof text, typeof isTrue);

Example 5: Operators

// Arithmetic and comparison operators\nlet a = 10;\nlet b = 20;\nlet sum = a + b;\nlet isEqual = a === b;\nconsole.log(\"Sum:\", sum, \"Are they equal?\", isEqual);
